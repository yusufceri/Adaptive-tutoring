# Adaptive STEM Learning Assistant

## Overview

Adaptive STEM Learning Assistant is an AI-powered educational platform designed to help students learn mathematics, physics, and introductory computer science through interactive, guided problem solving.

The project focuses on creating a more personalised learning experience compared to traditional chat-based educational tools. Instead of simply providing direct answers, the system encourages students to think through problems step-by-step using guided questioning and adaptive feedback.

---

## Problem

Many students struggle to access personalised STEM tutoring due to cost, scalability, and limited one-on-one educational support. Existing AI learning tools often:

- Lose context between sessions
- Provide answers too quickly
- Fail to adapt to different learning levels
- Do not effectively track learning progress

This project explores how AI systems can provide more structured and supportive learning experiences.

---

## Core Goals

The platform aims to:

- Support interactive STEM learning
- Adapt to student progress over time
- Encourage reasoning instead of answer memorisation
- Provide guided practice and feedback
- Improve accessibility to educational support

---

## Key Features

### Personalised Learning
The system keeps track of student progress and adjusts explanations and exercises based on demonstrated understanding.

### Guided Tutoring
Students receive hints, questions, and learning guidance rather than immediate solutions.

### Context-Aware Assistance
The platform maintains learning continuity across sessions to support long-term improvement.

### Knowledge Retrieval
Educational content is retrieved dynamically from curated learning resources to support accurate and contextual explanations.

### Progress Tracking
Students can review learning progress, topic familiarity, and practice history through a visual dashboard.
